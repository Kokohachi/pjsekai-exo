pjsekai-exoは、プロセカの創作譜面をプロセカ風の動画にするためのオープンソースのツールです。

=== 必要なもの ----------------------------------------------------------------

- AviUtl
  - 拡張編集
  - L-SMASH Works等の入力プラグイン

=== 使い方 --------------------------------------------------------------------

1. pjsekai-exo.exeを起動して、pedファイル（=Pjsekai Exo Data）と
   exoファイル（AviUtlのオブジェクトファイル）を作成します。
2. 拡張編集付きAviUtlの`/Plugins/script`フォルダに`@プロセカ.obj` を置きます。
3. 1334 x 750, 60fpsでプロジェクトを作成します。
4. `dist/譜面ID`内の`00_root.exo`はRoot、`01_main.exo`はシーン1...というように
   ファイルを読み込みます。
5. シーンを設定します。（下を参照）
6. オブジェクトの長さなどを調整してエンコードします。

----- シーンの対応表 ---------------------------------------------------------

------- Root -----------------------------------------------------------------

| オブジェクト    | シーン               |
| --------------- | -------------------- |
| Layer 1: 1..739 | シーン3（`背景用`）  |
| Layer 2: 1..208 | シーン2（`情報表示`）|
| Layer 2: 209..  | シーン1（`メイン`）  |

------- シーン3（情報表示） --------------------------------------------------

| オブジェクト    | シーン               |
| --------------- | -------------------- |
| Layer 5         | シーン3（`背景用`）  |

=== 仕組み -------------------------------------------------------------------

Rubyで譜面データを取得して、特殊なフォーマットに書き出しています。
そしてAviUtlのカスタムオブジェクト機能で、そのフォーマットを読み込み画像を配置しています。
フォーマットの詳細はGitHubのped_info.mdを参照して下さい。
